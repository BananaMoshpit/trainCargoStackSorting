A train cargo organizer using stacks and linked lists

An online compiler was used: https://tio.run/. For this, the code was all meshed into a single file, because I don't know how to add custom libraries to online compilers

tio.run is an inline interpreter supporting 681 languages and allows easy code usage without installing or downloading anything, thank you tio.run!

Here the input was added already, but for custom input just write it inside the input box and click play


This is meant to be a system of stack loading packages in trains, with the heaviest cars nearer to the cabine and heaviest packages by the bottom
Input should inform a the max amount of cars per train, the max weight of each car and a sequence of packages' weights Output is a line of cars each filled with a stack of packages, both from heaviest-to-lighter order
